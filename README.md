# AnimSnatcher

AnimSnatcher is a Roblox animation utility hub for capturing, managing, and playing animations with a clean UI.

## Features
- Steal animations from nearby players
- Play & stop animations
- Speed control
- Favorites system
- Emote tagging
- New animation badge
- Search & filters
- Pagination
- Draggable UI with minimize & restore

## Notes
- Single-file Lua script
- Designed for experimental / educational use

Enjoy.
